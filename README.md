#OMap Mapper
A map logger and visualization tool for orienteering maps. Setup a database for all your maps and browse them by map.

![Screenshot][1]

## Requirements
 - PHP5+
 - A MariaDB or MySQL database.
 
###Browser Support:
 - Opera
 - Internet Explorer 10/11
 - Chrome
 - Firefox(not recommended)
